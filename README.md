# GitHub-Template
A template for NSB/AppStudio projects

## Usage

This repository is a starting-point template for [NSB/AppStudio](https://www.nsbasic.com/) projects.

### Contents

- `GitHub-Template.project` – AppStudio project file (JavaScript, one empty form)
- `forms/Form1/Form1.json` – Form definition with no elements
- `forms/Form1/Form1.js` – JavaScript code file for Form1

### Getting Started

1. Click **Use this template** on GitHub to create a new repository from this template.
2. Open the project in NSB/AppStudio by loading `GitHub-Template.project`.
3. Rename the project file and update the `id` and `title` fields inside it to match your new project name.
4. Add forms and elements as needed.
